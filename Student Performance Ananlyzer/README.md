# Student Performance Analyzer

This project is a Python-based student management and performance analysis system developed while practicing programming and data analysis concepts.

The program allows users to add student records, calculate grades, analyze class performance, and store records in a text file.

## Features

- Add student records
- Automatic grade calculation
- Class average calculation
- Find top student
- Find lowest student
- Pass and fail statistics
- Save records to file
- Load previous records automatically
- Clear all records
- Menu-driven system

## Technologies Used

- Python

## Concepts Practiced

This project helped me practice:

- Functions
- Conditions
- Loops
- Lists
- File handling
- Data analysis
- Menu systems
- Structured programming

## Files

- `student_analyzer.py` → main program
- `students.txt` → stores student records

## How to Run

1. Install Python
2. Open the project folder in VS Code
3. Run the program using:

```bash
python student_analyzer.py
```

## Purpose of the Project

The purpose of this project was to improve my Python and data analysis skills by building a student performance system with multiple analytical features. While developing this project, I practiced working with structured data, file handling, functions, and basic statistical analysis.
