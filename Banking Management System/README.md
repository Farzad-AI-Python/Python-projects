# Banking Management System

A terminal-based Banking Management System developed using Python.

## Features

- Create Bank Accounts
- Deposit Money
- Withdraw Money
- Transfer Funds Between Accounts
- Check Account Balance
- Transaction History Tracking
- View All Accounts
- JSON-Based Data Storage
- Input Validation and Error Handling

## Technologies Used

- Python
- JSON
- File Handling
- Datetime Module

## How to Run

1. Open the project folder in VS Code
2. Run the following command:

```bash
python banking_system.py
```

## Sample Operations

- Create new customer bank accounts
- Deposit and withdraw money
- Transfer funds between users
- View balances and transaction history
- Store account records using JSON

## Project Purpose

This project was created to practice banking operations, transaction management, structured data storage, file handling, and real-world financial system logic using Python. It also helped improve understanding of user account management and transaction-based workflows commonly used in banking software systems.
