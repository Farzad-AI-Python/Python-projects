# Chat Application

A simple real-time Chat Application developed using Python socket programming and threading.

## Features

- Real-Time Messaging
- Multi-Client Communication
- Client-Server Architecture
- Socket Programming
- Multi-Threading Support
- Local Network Chat Simulation

## Technologies Used

- Python
- Socket Programming
- Threading

## How to Run

### Start the Server

Run the following command:

```bash
py chat_server.py
```

### Start the Clients

Open separate terminals and run:

```bash
py chat_client.py
```

## Sample Workflow

- Start the chat server
- Connect multiple clients
- Enter usernames
- Send and receive messages in real time
- Type `exit` to leave the chat

## Project Purpose

This project was created to practice networking concepts, socket programming, client-server communication, and multi-threading using Python. It demonstrates how real-time communication systems work in computer networks and software applications.
