# Face Recognition Attendance System

A Python-based Face Recognition Attendance System that uses computer vision to detect faces through a webcam and automatically record attendance.

## Features

- Real-Time Face Detection
- Webcam Integration
- Automatic Attendance Recording
- CSV-Based Attendance Storage
- Face Highlighting with OpenCV
- Date and Time Logging
- Attendance Record Viewer

## Technologies Used

- Python
- OpenCV
- CSV File Handling
- Computer Vision

## How to Run

1. Install the required library:

```bash
py -m pip install opencv-python
```

2. Run the program:

```bash
py attendance_system.py
```

## Sample Workflow

- Start the attendance system
- Enter student name
- Detect face using webcam
- Automatically mark attendance
- View attendance records

## Project Purpose

This project was created to practice computer vision, webcam processing, face detection, file handling, and attendance automation using Python and OpenCV. It demonstrates how image processing and real-time detection systems can be applied in educational and security-related applications.
