# AI Study Assistant

A simple AI-based Study Assistant chatbot developed using Python. The chatbot responds to study-related questions and provides basic educational guidance on topics such as AI, Python, exams, English learning, and productivity.

## Features

- Interactive Chatbot Interface
- Study and Learning Guidance
- AI and Python Information
- Exam Preparation Tips
- English Learning Suggestions
- Real-Time Date and Time Responses
- User Input Handling
- Simple Rule-Based AI Logic

## Technologies Used

- Python
- Conditional Statements
- Functions
- Datetime Module
- Text Processing

## How to Run

1. Open the project folder in VS Code
2. Run the following command:

```bash
py ai_study_assistant.py
```

## Sample Workflow

- Start the chatbot
- Ask questions related to study, AI, Python, exams, or English
- Receive automated responses
- Type `bye` to exit the chatbot

## Project Purpose

This project was created to practice basic artificial intelligence concepts, chatbot logic, user interaction, conditional programming, and text processing using Python. It demonstrates how rule-based conversational systems can respond to user input and simulate simple AI assistant behavior.
