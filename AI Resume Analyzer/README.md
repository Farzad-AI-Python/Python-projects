# AI Resume Analyzer

A Python-based Resume Analyzer that reads a resume text file and evaluates it based on skills, contact information, word count, and overall resume score.

## Features

- Read Resume Text Files
- Detect Technical and Soft Skills
- Identify Missing Recommended Skills
- Check Email and Phone Number
- Count Resume Words
- Generate Resume Score
- Provide Improvement Suggestions

## Technologies Used

- Python
- Regex
- File Handling
- Text Analysis

## How to Run

1. Open the project folder in VS Code
2. Run the following command:

```bash
py resume_analyzer.py
```

3. Enter the resume file name:

```txt
sample_resume.txt
```

## Sample Workflow

- Add resume content in a text file
- Run the analyzer
- Enter the file name
- View skills found, missing skills, contact details, word count, score, and suggestions

## Project Purpose

This project was created to practice text processing, resume screening logic, regular expressions, file handling, and basic AI-style analysis using Python. It helps demonstrate how software can be used to analyze documents and provide useful feedback.
