# Weather App

A Python-based Weather Application that fetches and displays real-time weather information for cities using an online weather service API.

## Features

- Real-Time Weather Information
- City-Based Weather Search
- Temperature Display
- Feels Like Temperature
- Humidity Information
- Wind Speed Display
- Weather Condition Detection
- Error Handling for Invalid Input and Connection Issues

## Technologies Used

- Python
- Requests Library
- API Integration
- JSON Data Handling

## How to Run

1. Install the required library:

```bash
py -m pip install requests
```

2. Run the program:

```bash
py weather_app.py
```

## Sample Workflow

- Enter a city name
- Retrieve live weather information
- View temperature, humidity, wind speed, and weather condition
- Search multiple cities
- Type `exit` to close the application

## Project Purpose

This project was created to practice API integration, JSON data processing, HTTP requests, user input handling, and real-time data retrieval using Python. It demonstrates how applications can communicate with online services to fetch and display dynamic information.
