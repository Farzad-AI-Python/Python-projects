# Advanced Calculator

This project is a command-line calculator developed in Python while learning programming fundamentals and problem solving. The calculator can perform different mathematical operations and also stores calculation history in a text file.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Power calculations
- Modulus calculations
- Save calculation history
- Load previous history automatically
- Clear history option

## Technologies Used

- Python

## Concepts Practiced

This project helped me practice:

- Variables
- User input
- Conditions
- Loops
- Functions
- Lists
- File handling

## Files

- `advanced_calculator.py` → main program
- `history.txt` → stores calculation history

## How to Run

1. Install Python
2. Open the project folder in VS Code
3. Run the program using:

```bash
python advanced_calculator.py
```

## Purpose of the Project

The purpose of this project was to strengthen my Python programming skills by building a calculator with features beyond a basic beginner project. While developing this calculator, I also learned how functions, loops, lists, and file handling work together in a practical application.
