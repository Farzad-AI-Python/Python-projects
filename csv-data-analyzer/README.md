# CSV Data Analyzer

This is a beginner Python data analysis project created using pandas and matplotlib.

## Project Overview

The project reads sales data from a CSV file and performs simple data analysis using Python.

It calculates:
- Total sales for each product
- Overall revenue
- Best selling product
- Category-wise sales

The project also generates a bar chart to visualize sales by category.

---

## Features

- Read CSV sales data
- Calculate total sales
- Find total revenue
- Find best selling product
- Analyze category sales
- Export updated CSV data
- Generate sales chart

---

## Technologies Used

- Python
- Pandas
- Matplotlib

---

## Files Included

- `main.py` → Main Python program
- `sales_data.csv` → Original dataset
- `updated_sales_data.csv` → Processed dataset
