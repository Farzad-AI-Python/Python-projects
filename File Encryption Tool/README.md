# File Encryption Tool

A Python-based File Encryption Tool that securely encrypts and decrypts files using cryptography techniques.

## Features

- Generate Secure Encryption Keys
- Encrypt Text Files
- Decrypt Encrypted Files
- Secure File Processing
- Symmetric Encryption using Fernet
- Error Handling and Validation

## Technologies Used

- Python
- Cryptography Library
- File Handling

## How to Run

1. Open the project folder in VS Code
2. Install the required library:

```bash
py -m pip install cryptography
```

3. Run the program:

```bash
py file_encryption.py
```

## Sample Workflow

- Generate an encryption key
- Encrypt a text file
- Decrypt the encrypted file
- Restore original file content securely

## Project Purpose

This project was created to practice file security, encryption and decryption techniques, secure key management, and cybersecurity-related programming concepts using Python.
