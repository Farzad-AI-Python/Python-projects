# Expense Tracker

A terminal-based Expense Tracker developed using Python. This project helps users record expenses, view spending history, calculate total expenses, summarize spending by category, and delete records.

## Features

- Add New Expenses
- View Expense Records
- Calculate Total Spending
- Category-Based Spending Summary
- Delete Expense Records
- JSON-Based Data Storage
- Input Validation and Error Handling
- Automatic Date and Time Tracking

## Technologies Used

- Python
- JSON
- File Handling
- Datetime Module

## How to Run

1. Open the project folder in VS Code
2. Run the following command:

```bash
py expense_tracker.py
```

## Sample Workflow

- Add an expense with title, amount, and category
- View all saved expenses
- Check total spending
- View category-wise spending summary
- Delete an expense record if needed
- Store records automatically using JSON

## Project Purpose

This project was created to practice file handling, data storage, user input validation, expense management logic, and basic financial tracking using Python. It demonstrates how programming can be used to organize personal finance records and summarize spending behavior.
